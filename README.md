# Proyecto de Paradigmas de Programación #
En este proyecto, realizamos un sistema de biblioteca simple que se encargue de administrar esta pequeña biblioteca de forma sencilla y segura.
Debe evitar préstamos imposibles, libros duplicados y usuarios inexistentes.
Y utilizará 3 archivos en los cuales se almacenará toda información:

    libros.json

    usuarios.json

    prestamos.json

Dado que es un proyecto principiante, puede que cuente con errores que se irán reparando de a poco.
Siendo así que las primeras versiones solo incluirán todos los procesos en terminal, y las más lejanas implementen interfaces gráficas.
